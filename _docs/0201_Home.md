# Tela Home (0201)

## Propósito
Listar os itens da tabela `todos` do Supabase com fallback para mocks, exibindo um painel de gerenciamento do transporte escolar municipal de Arapongas com layout premium glassmórfico.

## Rota
- `/`

## Componentes Utilizados
- **Server Component Page:** `app/page.tsx`
- **Server Client Helper:** `utils/supabase/server.ts`
- **Root Layout:** `app/layout.tsx`

## Contrato de Dados / Tipos
```ts
interface Todo {
  id: number;
  name: string;
}
```
