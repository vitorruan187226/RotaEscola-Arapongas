# Tela Home (0201) — Landing Page

## Propósito
Landing page pública e institucional do sistema RotaEscola Arapongas. Apresenta o sistema com Hero, Benefícios (Pais), Recursos (Bento Motoristas), Tutorial de Funcionalidade e links de app store. Todo o design foi refatorado utilizando Tailwind CSS, alinhando-se aos visuais do projeto base (Stitch HTML export).

**Nota:** Esta é a tela raiz do sistema. O orchestrator em `app/page.tsx` chama seções modulares e não contém CSS inline.

## Rota
- `/` → `app/page.tsx`

## Modo de Renderização
- `app/page.tsx` — Server Component leve (orchestrator).
- Componentes internos como `HeaderSection` podem usar `'use client'` para estados de UI.

## Componentes e Dependências
- `components/landing/HeaderSection.tsx`
- `components/landing/HeroSection.tsx`
- `components/landing/BeneficiosSection.tsx`
- `components/landing/RecursosSection.tsx`
- `components/landing/CtaBannerSection.tsx`
- `components/landing/SuporteSection.tsx`
- `components/landing/FooterSection.tsx`
- Ícones via `lucide-react`.

## Seções da Página
| Seção | Descrição |
|---|---|
| Header | Logo, nav links (desktop), CTA "Acessar Sistema", menu mobile. |
| Hero | Hero image (`hero-bus.jpg`), textos, CTA secundário e CTA "Primeiro Acesso". |
| Benefícios | Seção "Para os Pais": 3 cards mostrando Rastreamento, Avisos de Chegada e Histórico. |
| Recursos | Seção "Gestão Eficiente" (Motoristas): Bento Grid (`dashboard-mockup.jpg`), CTA para `/login?role=motorista`. |
| Cta Banner | "Simples como deve ser": Step-by-step visual para onboarding. |
| Suporte | "Pronto para dar mais segurança...": Badges visuais do Google Play e App Store. |
| Footer | Logo, links úteis, direitos autorais, redes sociais (ícones share, mail). |

## Contrato de Dados / Tipos
Tela estática baseada nos componentes React da pasta `components/landing`. Sem integrações externas ou SSR fetching, direciona usuários aos portais em `/login`.
