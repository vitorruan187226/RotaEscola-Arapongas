# Tela Home (0201) — Landing Page

## Propósito
Landing page pública e institucional do sistema RotaEscola Arapongas. Porta de entrada para pais, motoristas e servidores da Secretaria de Educação. Apresenta o sistema com Hero, Vantagens, Recursos, CTA de recadastro e Suporte.

**Nota:** Esta é a tela raiz do sistema — não é uma tela de teste. Deve ser preservada e evoluída conforme o roadmap.

## Rota
- `/` → `app/page.tsx`

## Modo de Renderização
- `'use client'` — Contém estado React (`useState`) para o menu hamburguer mobile.

## Componentes e Dependências
| Módulo | Uso |
|---|---|
| `next/link` | Navegação client-side |
| `lucide-react` | Todos os ícones (Bus, ShieldCheck, MapPin, QrCode, FileText, WifiOff, ArrowRight, Users, BarChart3, Smartphone, CheckCircle, Menu, X, Phone, Mail, Building2, ChevronRight, Star, Zap) |
| `@/components/ui/button` | Botões shadcn/ui com variantes `yellow`, `navy`, `white-outline` |
| `useState` | Controle do menu mobile (hamburguer) |

## Seções da Página
| Seção | ID/Elemento | Descrição |

| CTA Recadastro | `<section class="lp-cta-banner">` | Banner navy com chamada para recadastro anual 2025 |
