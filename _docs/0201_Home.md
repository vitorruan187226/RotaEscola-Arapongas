# Tela Home (0201)

## Propósito
Landing page institucional da rota `/`. Apresenta o produto RotaEscola Arapongas para pais e servidores municipais. Exibe Hero, seção de Recursos e CTA com links para as áreas de login.

**Nota:** Esta é a tela raiz do sistema — não é uma tela de teste. Ela deve ser preservada e evoluída conforme o roadmap.

## Rota
- `/` → `app/page.tsx`

## Componentes Utilizados
- **Server Component Page:** `app/page.tsx`
- **Root Layout:** `app/layout.tsx`
- **Ícones:** `lucide-react` (Bus, ShieldCheck, MapPin, QrCode, ClipboardList, ArrowRight)

## Dependências de Estilo
- Todas as classes CSS estão em `app/globals.css` (`@layer components`)
- **Não há mais `<style>` inline** neste componente (migrado na Fase 3 do Item 1)
- Animações utilizadas: `.animate-slide-up` (hero-content), `.animate-fade-in` (hero-visual)

## Seções da Página
| Seção | Classe raiz | Descrição |
|---|---|---|
| Navbar | `.navbar` | Logo, subtítulo, botão de acesso |
| Hero | `.hero-section` | Badge, H1, parágrafo, 2 CTAs, card visual |
| Features | `.features-section` | Grid de 3 cards com ícones |
| CTA | `.cta-section` | Card navy com 2 botões de ação |
| Footer | `.footer` | Dados de contato da Secretaria |
